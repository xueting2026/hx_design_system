页型 Page Types

🔐 登录页
身份认证，不承载业务信息
🏠 首页
概览待办任务，快速跳转各业务模块
🧩 功能选择页
子功能≥3个时作为二级入口集合
📋 列表页
同类任务集合，筛选/搜索/批量操作（最核心）
✏️ 操作页
针对特定对象录入信息并确认提交
💬 弹窗/面板
轻量级操作（≤2步），不需要独立页面

决策树

用户要做什么？ ├── 认证身份 → 登录页 ├── 概览全局 → 首页 ├── 选子功能(≥3个) → 功能选择页 ├── 浏览/筛选/操作列表 → 列表页（最常用） ├── 录入信息并提交 │ ├── 复杂(多字段/上传) → 操作页 │ └── 简单(1-2个字段) → 弹窗/面板 └── 查看详情 → 详情页
