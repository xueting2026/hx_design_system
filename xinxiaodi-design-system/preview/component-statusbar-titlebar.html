使用规则
状态栏 + 标题栏的组合与显示规则
1. 分组：状态栏 + 标题栏 打包为一组 仅 demo 用
  • 两者用 <div class="demo-topbar"> 容器包裹为一组
  • 该组仅用于 demo / 原型预览，还原真机顶部效果
  • 开发时系统（小程序 / App）自带顶部，无需实现
开发隐藏方式：.demo-topbar { display:none }，或加 .demo-topbar--hidden，或直接删除整个 .demo-topbar 容器。
2. 状态栏：始终存在
  • demo 中状态栏一直显示（9:41 + 系统图标）
  • 配色跟随页面：白底页用 statusbar--dark，渐变头部页用 statusbar--light
3. 标题栏：按当前页面决定，由三部分组成
  • 返回按钮：仅二级及以下页面有，点击返回上一级；一级页面（首页、登录页）无返回按钮
  • 标题：始终存在，按当前页面命名（如「首页」「待到库」）；登录页标题为「信小递」。标题始终相对整栏居中，不管左右两侧按钮功能增减都保持居中
  • 胶囊按钮：小程序平台始终提供，固定在右侧，距右 6px，宽度自适应（87×32）
